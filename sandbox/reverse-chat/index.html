Reverse Mode + Groups

Chat-style messaging UI using reverse: true with groups plugin (sticky: false). Demonstrates iMessage-style inline date headers that automatically organize messages by date. The list starts scrolled to the bottom, loads older messages at the top with preserved scroll position, and auto-scrolls when sending new messages.

Reverse mode with groups plugin supports both inline headers (sticky: false - iMessage/WhatsApp style) and sticky headers (sticky: true - Telegram style). This example uses inline headers. Try changing sticky: false to sticky: true in the code to see the difference! Items stay in chronological order (oldest = index 0). appendItems() auto-scrolls when at bottom; prependItems() preserves scroll position. ⚡
